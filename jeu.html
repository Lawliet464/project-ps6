Comment jouer ?

🃏 1. Le memory
Clique sur une carte pour la voir. Puis clique sur une autre. Si elles sont pareilles, elles restent ouvertes !

🦷 2. L'association
Mets les objets dans le bon ordre. Glisse-les ou clique dessus puis clique sur une case.

💡 Le bouton Aide
Appuie dessus si tu es bloqué, ça t'aidera !

1. Le memory
Retournez les cartes pour trouver les paires identiques. Après un certain nombre d'erreurs, une aide visuelle s'affiche automatiquement.

2. L'association
Associez les cartes dans le bon ordre selon le thème. Le bouton "Vérifier" apparaît une fois toutes les cases remplies. Un message d'erreur détaillé indique les mauvais placements.

💡 Bouton Aide
Pour le patient : mise en surbrillance. Pour vous : explication complète de l'erreur avec type (mauvais thème ou mauvais ordre).
