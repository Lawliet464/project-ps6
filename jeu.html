Comment jouer ?

1. Le memory
Cliquez sur n'importe quelle carte pour la révéler. Retournez une seconde carte. Si les deux correspondent, elles restent visibles.

2. L'association
Associez les cartes du memory dans le bon ordre (dans la bonne cellule numérotée) selon le thème donné. Changez votre réponse à tout moment avant de valider : vous pouvez déplacer l'élément que vous souhaitez dans une cellule vide ou la zone des éléments disponibles. Les thèmes vous permettent de vous rappeler des gestes et objets du quotidien.

3. Le bouton 💡 Aide
Il vous donne un indice si vous êtes bloqué.
